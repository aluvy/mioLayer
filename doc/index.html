mioLayer

Rule

스크롤 영역

  • 스크롤 되는 영역에 data-scroll-body 속성을 부여한다.
  • 모달 오픈 시 scrollDisabled 옵션에 따라 스크롤 영역이 비활성화 된다.
이 예제에서는 body의 첫 번째 자식인 #wrap이 스크롤된다.
EX) <div id="wrap" data-scroll-body>

모달 아이디

  • 아이디는 페이지 내에 유일해야 한다.
    EX) MIOLAYER
  • 버튼의 아이디는 ${아이디-BTN}로 한다.
    EX) MIOLAYER-BTN
  • 생성된 모달의 ID는 #{아이디}-MD로 자동 부여된다.
    EX) #MIOLAYER-MD
  • 생성된 모달의 ID가 없을 경우 (마크업 없이 호출 시) .miolayer에 ID값으로 #MIOLAYER-MD가 부여된다.

모달 내 버튼 이벤트

  • 모달 내 기능 버튼에는 data-action 속성을 부여한다.
    • close : 모달 닫기
    • confirm : 모달옵션 중 fn.confirm() 함수 실행
    • cancel : 모달옵션 중 fn.cancel() 함수 실행
<button type="button" class="btn secondary" data-action="close">닫기</&button>
<button type="button" class="btn secondary" data-action="confirm">확인</&button>
<button type="button" class="btn secondary" data-action="cancel">취소</&button>

$.mioLayer({
  type: 'layer',
  title: '알림',
  displayTitle: true,
  displayCloseButton: true,
  dimAction: true,
  content:'#MIOLAYER',
  displayBottomButtons: true,
  buttons: [
    ``,
    ``,
    ``,
  ],
  fn: {
    confirm(ins) {
      // data-action="confirm" 클릭 시 실행
      ins.fn.close(ins);  // 해당 모달창 닫기
    },
    cancel(ins) {
      // data-action="cancel" 클릭 시 실행
    }
  },
  scrollDisabled: 'disabled'
});

Options

  • type: 모달 타입 - [ 'layer' | 'confirm' | 'alert' ] - default: 'layer'
  • title: 모달 타이틀 - default: "알림"
  • displayTitle: 타이틀 노출 여부 - [ true | false ] - default: true
  • displayCloseButton: 닫기 버튼 노출 여부 - [ true | false ] - default: true
  • dimAction: dim 클릭 시 모달창 닫기 실행 여부 - [ true | false ] - default: true
  • content: 모달 컨텐츠 - [ "element ID" | String ]
    • element id: 모달 엘리먼트 ID
    • String: 텍스트
  • displayBottomButtons: 하단 버튼 영역 노출 여부 - [ true | false ] - default: false
  • buttons: 하단 버튼 영역 버튼 목록 - [ array ]
    • 버튼 샘플: <button type="button" class="btn secondary" data-action="close">닫기</&button>
  • fn: 콜백함수
    • confirm(): data-action="confirm" 클릭 시 실행, parameter로 instance를 받는다.
    • cancel(): data-action="cancel" 클릭 시 실행, parameter로 instance를 받는다.
  • scrollDisabled: 스크롤 비활성화 여부 - [ true | false ] - default: true
fn.confirm(), fn.cancel() 함수에서 모달창 닫기를 실행하고 싶은 경우 각 함수 내에서 ins.fn.close(ins) 를 호출한다.
